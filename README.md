# Inventory Management System

This project is a simple organizational tool, that is written in C++. It allows the user to add, view and delete items from an inventory list. Each of the items in the list contains a name, quanity and price. Usiing classes and inheritance, with virtual function. I wanted to use a dynamic memory and file the input and outputs to store the inventory data. 

## Instructions for Build and Use

[Software Demo]

Steps to build and/or run the software:

1. Open the project folder in Visual Studio Code
2. Open terminal in VSC
3. Compile the program I used: g++-15 inventory.cpp -o inventory
4. Run the program: ./inventory

Instructions for using the software:

1. Select option 1 to add a new item (enter name, quantity, and price)
2. Select option 2 to view all items in the inventory
3. Select option 3 to delete an item by its index
4. Select option 4 to save and exit the program
